# Movie Data Analysis CLI (Python)

## Description
A command-line Python application that analyzes movie data from a CSV file and generates reports based on year, genre, and votes.

## Features
- Year-based report
- Genre-based statistics
- Vote-based ranking
- CLI interface using argparse
- Handles invalid/missing data

## How to Run

```bash
python main.py -r 1903
python main.py -g Comedy
python main.py -v 1000
